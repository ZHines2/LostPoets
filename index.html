LOST POETS — Help & Tutorial

    
    
      
  • Movement: WASD or Arrow Keys
    • 
      
  • Four Tools (modes of interaction):
    • 
      
  •   • 1=♠Spade - Dig/harvest (hands → augmented with tools)
    • 
      
  •   • 2=♣Club - Combat (fists → augmented with weapons)
    • 
      
  •   • 3=♥Heart - Communicate/meditate (internal functions)
    • 
      
  •   • 4=♦Diamond - Craft at workbench (choose recipes!)
    • 
      
  • Use Tool: Shift+Direction or E (facing dir)
    • 
      
  • Actions: G=Pick up, P=Place torch, R=Rest, O=Observe, .=Wait, I=Inventory
    • 
      
  • Crafting System (16 recipes!):
    • 
      
  •   • Select Diamond tool (press 4) then use at workbench (Shift+Dir or E)
    • 
      
  •   • Press 1-9 to select recipe, 0 to cancel
    • 
      
  •   • Press - / = to scroll between recipe pages
    • 
      
  •   • Craft Portable Workbench to craft anywhere!
    • 
      
  •   • New recipes unlock as you craft more items
    • 
      
  • Inventory System:
    • 
      
  •   • Tools augment abilities (Axe=chop trees, Pickaxe=mine ore)
    • 
      
  •   • Press I to view collected items
    • 
      
  •   • Multiple tools stack - best bonus applies
    • 
      
  • Other: V=Toggle visibility, Q=Reload, H/K=Toggle help
    • 
    

    
Press H or K to close
